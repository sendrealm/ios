# Sendrealm iOS SDK Changelog

## 0.1.2 - 2026-07-02

### Changed
- Increase native pending queue retention from 100 to 1000 items.

## 0.1.1 - 2026-06-28

### Changed
- Point README users to the public Sendrealm docs.
- Update the reported SDK version.

## 0.1.0 - 2026-06-28

### Added
- Initial native Swift APNs SDK with device initialization, APNs token registration, permission requests, SDK state, identity, subscription controls, tags, event tracking, offline queues, foreground/open handling, payload normalization, APNs environment support, and rich notification helper.

### Migration Notes
- This is the first public iOS SDK baseline.
